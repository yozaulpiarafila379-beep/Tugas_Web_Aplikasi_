1. Class User (Login Sistem)

  • Class User merepresentasikan data pengguna yang memiliki hak akses terhadap sistem informasi pendataan barang di gudang.
  • Atribut: username, password, dan levelUser — digunakan sebagai informasi autentikasi.
  • Constructor User() — melakukan inisialisasi data ketika objek User dibuat.
  • Method login() — melakukan validasi username dan password sebelum mengakses sistem.
  • Fungsi main() — membuat objek User dan menjalankan proses autentikasi.

Input / Proses / Output:

  • Input: Username, password, level akses pengguna.
  • Proses: Membuat objek User → menyimpan data → menjalankan login() → validasi data.
  • Output: Informasi login berhasil / gagal + data pengguna yang masuk.

Kode Dart:

// ==========================================
                // Nama 1 : Yoza Ulpia Rafila | NIM : 251410004
                // Nama 2 : Rifqi Prayoga      | NIM : 251410005
                // Kelas  : SI2A | Sistem Informasi Pendataan Barang Gudang
                // ==========================================
                // Class User: Mengelola autentikasi pengguna sistem
                class User {
                // --- Properti / Atribut ---
                String username;   // Nama pengguna untuk login
                String password;   // Kata sandi untuk autentikasi
                String levelUser;  // Level akses: Admin Gudang, Staff, dll.

                // --- Constructor: Inisialisasi data pengguna ---
                User(this.username, this.password, this.levelUser);

                // --- Method login(): Validasi kredensial pengguna ---
                void login() {
                if (username == "admin" && password == "12345") {
                print("✅ Login berhasil");
                print("👤 User  : $username");
                print("🔑 Level : $levelUser");
                } else {
                print("❌ Login gagal — periksa username & password");
                }
                }
                }

                // --- Fungsi main(): Entry point program ---
                void main() {
                // Membuat objek User dengan data login
                var user1 = User("admin", "12345", "Admin Gudang");
                user1.login(); // Menjalankan proses autentikasi
            }

2. Class Barang (Data Barang)

  • Class Barang merepresentasikan data barang dalam sistem informasi gudang.
  • Atribut: kodeBarang, namaBarang, kategori, merk, jumlahStok, satuan, kondisi.
  • Constructor Barang() — inisialisasi data barang.
  • Method tampilkanBarang() — menampilkan detail barang.
  • Method tambahStok() — menambah stok (barang masuk).
  • Method kurangiStok() — mengurangi stok (barang keluar).

Kode Dart:

// Class Barang: Merepresentasikan setiap item di gudang
                class Barang {
                // --- Properti ---
                String kodeBarang;  // Kode unik identifikasi barang
                String namaBarang;  // Nama lengkap barang
                String kategori;    // Kategori / jenis barang
                String merk;        // Merek barang
                int jumlahStok;      // Jumlah stok tersedia
                String satuan;      // Satuan: Unit, Box, Pcs, dll.
                String kondisi;     // Kondisi: Baik, Rusak, Expired

                // --- Constructor ---
                Barang(this.kodeBarang, this.namaBarang, this.kategori,
                this.merk, this.jumlahStok, this.satuan, this.kondisi);

                // --- Method: Menampilkan detail barang ---
                void tampilkanBarang() { /* ... */ }
                // --- Method: Menambah stok ---
                void tambahStok(int jumlah) { jumlahStok += jumlah; }
                // --- Method: Mengurangi stok ---
                void kurangiStok(int jumlah) { jumlahStok -= jumlah; }
            }
